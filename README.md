# 🌟 Leo Club of Hopeville Website

Official website for the **Leo Club of Hopeville**, a youth-led service organization proudly sponsored by the **Lions Club of Chennai Aavarampoo**.

The website showcases the club's mission, community initiatives, leadership, impact stories, and events through a modern, responsive, and user-friendly interface.

---

## ✨ Features

* Modern and responsive UI
* Home, About, Activities, Gallery, Team, and Contact pages
* Impact Stories highlighting community initiatives
* Parent Club information
* Instagram integration
* Contact form powered by Web3Forms
* Mobile-friendly design
* Optimized for fast performance with Vite

---

## 🌍 About the Club

Guided by the motto **"Serve Kindness,"** the Leo Club of Hopeville empowers young individuals to become compassionate leaders through meaningful community service.

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd leo-club-hopeville-website
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 📦 Production Build

```bash
npm run build
```

---

## 👀 Preview Production Build

```bash
npm run preview
```
