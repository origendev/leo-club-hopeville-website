# # Leo Club of Hopeville Website

Official website for the Leo Club of Hopeville.

## Features

- Responsive modern UI
- Home, About, Activities, Gallery and Team pages
- Impact Stories
- Parent Club information
- Contact form powered by Web3Forms
- Instagram integration
- Mobile-friendly design

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- Web3Forms

## Installation

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
